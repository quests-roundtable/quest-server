## Getting Started


### Prerequisites

* Java: 17.0.1 or later
* Maven: 3.8.2 or later


<!-- USAGE EXAMPLES -->
### Installation and Run

-> Maven install
   ```sh
   mvn clean install
   ```

-> Run the server
   ```sh
   ./mvnw spring-boot:run
   ```
The api runs at localhost:9090/.

The websocket url: ws://localhost:9090/ws.
